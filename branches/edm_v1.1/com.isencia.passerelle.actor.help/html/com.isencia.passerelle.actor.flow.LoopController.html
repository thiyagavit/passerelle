LoopController

com.isencia.passerelle.actor.flow.LoopController

An actor that can be configured with a loop count N, then forwards a received input message N times, and does this in a synchronized way. I.e. the loop in the model can send a feedback msg to trigger this loop controller to send the data one more time, for the next loop iteration.
