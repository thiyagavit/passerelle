ErrorCatcher

com.isencia.passerelle.actor.error.ErrorCatcher

An actor that extracts the ManagedMessage from ErrorMessages received on its input port, and sends it out. The error can optionally be logged. This is useful to link to an error-output of one or more actors, to implement an "ignore error" behaviour, combined with continued sequence operations "as-if-everything-was-OK". In this respect it differs from the plain DevNullActor, which discards the received msg and thus effectively interrupts the sequence. If the received message is not an ErrorMessage, or it does not contain a ManagedMessage as the error's context, it is just forwarded to the output as is.
